关于本项目

和栗薰子QQ机器人是一个基于C#开发的非盈利第三方项目，使用.NET技术栈和Mirai框架实现。该项目由粉丝创作，旨在为《熏香花朵凛然绽放》的爱好者提供有趣的聊天体验。

薰子机器人接入了DeepSeek的API，具备强大的AI对话能力，能够理解自然语言并进行智能回复。无论是日常闲聊、问题解答还是娱乐互动，薰子都能成为您的贴心助手。

使用方法

注意：由于个人开发者的局限，此机器人仅作为QQ频道机器人使用，无法回复普通QQ群消息。

使用手机QQ扫描二维码添加

简单三步添加机器人：

  1. 使用手机QQ扫描上方二维码
  2. 点击"添加到频道"按钮
  3. 选择您已创建的QQ频道完成添加

或直接点击链接：https://qun.qq.com/qqweb/qunpro/jump?id=robot-share&robot_appid=102805649
